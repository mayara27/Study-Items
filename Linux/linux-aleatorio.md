Manual do comando:

```
man ls (qualquer comando)
```
Ver a data do sistema:

```
date
```

Ajuda:

```
comando --help
```

Locatizar arquivo no sistema: 

```
locate nome_do_arquivo
```

Atuallizar bd:

```
sudo updatedb
```

Mostra o local do arquivo que vou executar se eu der o comando `code` por exemplo: 

```
which code
```

Saida:

```
/usr/bin/code
```
