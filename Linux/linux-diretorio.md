# Diretórios

Mostra o diretório que estou:
```
pwd
```
Lista os diretórios que tenho:

```
ls

ls -l #mostra destalhes do dir

ls -la #mostra até os dir invisiveis
```

Mudar de diretório:

```
cd diretorio
```

Criar diretorio: 

```
mkdir diretorio
```

Copiar diretorio:

```
cp -r diretorio
```

Remover diretorio:

```
rmdir diretorio
```
Remover diretorio com arquivos dentro:

```
rm -r diretorio
```
