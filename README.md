## :wrench: Configuração
(Configuração necessária para rodar o projeto)

No terminal clone o projeto:

```
git clone https://github.com/mayara27/Study-Items
```
Entre na pasta

```
cd pasta
```
Instale as dependencias

```
yarn install
```
